The components of a web application

A web application consists of clients and a web server. The clients are computers, tablest, and mobile devices that use the web applicaiton. They access teh web pages through web browsers. The web server holds the files that make up the web application.

How Static web pages are processed

A static web page is one that doesn't change each time it is requested. This type of web page is sent directly from the web server to the web browser when the browser requests it.

How Dynamic web pages are processed

A dyanmic web page is a page that is created by a program or script on the web server each time it is requested. This program script is ececuted by an applicaiton server based on the data that is sent along with the HTTP request.

How JavaScript is used for client-side processing

Unlike the server-side processing done for dynamic web pages, JavaScript is a scripting language that allows for client-side processing. JavaScript can be used to modify elements on web page without having to use server side processing.

JavaScript is often used to do the following on the client side:

  • Data Validation
  • Image swaps and rollovers
  • Controling slide shows
